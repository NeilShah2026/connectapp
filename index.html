EVDictionary GUI Version !

EVDictionaryTM là một ứng dụng desktop được viết bằng Java nhằm mục đích đem đến một phiên bản từ điển Anh Việt miễn phí dễ sử dụng cho mọi người.

Tính năng

Tra cứu

Với cơ sở dữ liệu hơn 109 nghìn từ, bạn có thể tra cứu nghĩa của hầu hết các từ tiếng Anh thường dùng.

Chỉnh sửa

Có thể sửa từ, thêm từ, xoá từ nếu từ điển xảy ra sai sót hoặc chưa đáp ứng được nhu cầu của bạn.

Google Translate

Ngoài cơ sở dữ liệu của ứng dụng, bạn còn có thể tra cứu từ ngữ thông qua Google Translate.

Phát âm

Ứng dụng có tích hợp API text to speech giúp bạn có thể nghe cách phát âm của một từ nào đó.

Công nghệ sử dụng

JavaFX

Xây dựng giao diện đồ hoạ người dùng dựa trên Swing và Java2D

CSS

Sử dụng ngôn ngữ CSS để trình diễn một số hiệu ứng thiết kế UI

JFoenix

Implement phong cách thiết kế Google Material Design.

SQLite JDBC

Driver giúp ứng dụng kết nối vói cở sở dữ liệU SQLite.

JSON in Java

Xử lí kết quả trả về dưới dạng JSON khi request tới Google Translate.

GoogleTTS

API tích hợp giúp chuyển dạng text sang giọng nói.

Vấn đáp





Mời thầy và các bạn cho câu hỏi và thảo luận.

EVDictionary được viết trên OpenJDK phiên bản 15, sử dụng IDE là IntelliJ IDEA Ultimate Edition 2020.3.

Như đã trình bày ở trên, chương trình sử dụng các thư viện như JavaFX, JFoenix, JSON in Java, GoogleTTS, driver SQLite JDBC.

Toàn bộ dữ liệu về từ tiếng Anh, nghĩa tiếng Việt, phát âm, loại từ,... được lưu trong file database duy nhất.

Bằng cách kết nối với database thông qua SQLite JDBC, chương trình nạp vào các nội dung từ database và lưu vào một cấu trúc dữ liệu là HashMap.

API Google Translate là hàng bản quyền, có trả phí. Nên chức năng sử dụng Google Translate thực chất không phải gọi qua API là mà request hẳn tới bộ dịch, kết quả trả về dưới dạng JSON, nên thư viện JSON in Java sẽ xử lí và trả về nghĩa tiếng Việt cuối cùng.

Ở một số button và pane, do thiết kế theo hướng Google Material Design nên ngoài các components được cung cấp sẵn bởi JFoenix, chương trình có sử dụng CSS để đồng bộ thiết kế này.
